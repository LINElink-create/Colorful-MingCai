# 明彩隐私政策

生效日期：2026-04-05

本隐私政策适用于“明彩（Colorful Reader）”浏览器扩展及其配套后端服务。本政策说明我们会处理哪些数据、为什么处理、如何存储和共享这些数据，以及你可以如何控制自己的数据。

如果你不同意本政策中的任何内容，请停止使用本扩展或相关服务。

## 1. 我们是谁

“明彩（Colorful Reader）”是一款用于网页划词高亮、翻译、导入导出、账号登录和云同步的浏览器扩展。

## 2. 本政策适用范围

本政策适用于以下产品和功能：

- 浏览器扩展本体
- 扩展使用的明彩后端接口
- 账号注册、登录、邮箱验证和云同步功能
- 通过扩展触发的文本翻译能力

本政策不适用于你主动访问的第三方网站本身，也不适用于第三方翻译服务或邮件服务提供商各自的独立隐私政策。

## 3. 我们处理哪些数据

我们只在实现扩展功能所必需的范围内处理数据。

### 3.1 保存在你的浏览器本地设备中的数据

当你使用本扩展时，扩展会在浏览器本地存储中保存以下数据：

- 你创建的网页高亮内容与摘录信息
- 与高亮关联的网页地址、网页标题、锚点定位信息、颜色、备注、创建时间和更新时间
- 翻译偏好设置，例如源语言、目标语言、默认翻译服务和是否自动翻译
- 扩展连接后端服务所需的登录状态信息，例如访问令牌和刷新令牌
- 云同步状态信息，例如最近同步时间、同步的站点数和高亮数

这些数据默认保存在你的本地浏览器环境中，除非你主动启用账号和云同步功能，否则不会自动上传到明彩服务器。

### 3.2 当你主动使用翻译功能时处理的数据

当你点击翻译按钮或触发后端翻译能力时，我们可能会处理以下数据：

- 你主动选中的文本内容
- 你设置的源语言、目标语言和默认翻译服务
- 当前页面的 URL 和页面标题
- 扩展版本号等基础客户端上下文

这些数据会发送到明彩后端，用于完成翻译请求。根据你的配置，明彩后端可能进一步将必要的翻译内容转发给以下服务之一：

- 有道翻译服务
- 你主动配置的 OpenAI 兼容翻译接口

明彩后端当前还会保存最小化的翻译请求记录，用于运行维护和防滥用控制，例如：

- 翻译服务提供方标识
- 源语言和目标语言
- 请求文本的哈希值，而不是完整原文日志
- 字符数、响应大小、耗时、状态码和错误码
- 哈希化后的 IP 信息和 User-Agent 信息

### 3.3 当你主动注册、登录或启用云同步时处理的数据

如果你选择使用账号系统，我们会处理以下数据：

- 电子邮箱地址
- 显示名称
- 你输入的密码
- 登录会话和刷新令牌
- 邮箱验证状态和邮箱验证令牌

你的密码不会以明文形式存储在数据库中，而是以哈希后的形式保存。

### 3.4 当你主动启用云同步时上传的数据

如果你登录并执行云同步，我们会将你选择同步的高亮数据上传到明彩后端，包括：

- 页面 URL
- 页面标题
- 高亮摘录内容
- 备注内容
- 锚点定位快照
- 导出或同步时间等结构化元数据

这些数据用于在同一账号下恢复、同步和管理你的网页标注内容。

### 3.5 当你主动保存个人翻译服务配置时处理的数据

如果你在账号体系下保存个人翻译服务配置，我们可能会处理你主动提交的配置，例如：

- 有道应用密钥
- OpenAI 兼容接口地址
- OpenAI 兼容接口密钥
- 模型名称

这些数据仅用于实现你主动启用的翻译能力，不会用于广告或画像用途。

### 3.6 当你主动使用邮箱验证时处理的数据

如果你注册账号或请求重发验证邮件，我们会处理以下数据：

- 你的邮箱地址
- 邮箱验证令牌及其状态
- 发信时间、验证时间和过期时间等必要元数据

验证邮件属于事务性邮件，仅用于完成账号验证。

## 4. 我们如何使用这些数据

我们处理数据的目的仅限于以下范围：

- 在页面中创建、恢复、删除和导出你的高亮与备注
- 保存你的本地配置和使用偏好
- 为你提供文本翻译能力
- 创建和维护账号登录状态
- 发送邮箱验证邮件并确认邮箱归属
- 在你主动启用时执行云同步
- 维护服务安全、诊断错误、限制滥用和改进稳定性

我们不会将你的数据出售给第三方，也不会将你的数据用于个性化广告投放。

## 5. 浏览器权限使用说明

本扩展当前申请以下主要权限，这些权限仅用于实现对应功能：

- `storage`：保存本地高亮、设置、登录状态和同步状态
- `contextMenus`：提供右键菜单中的高亮和取消高亮操作
- `downloads`：导出你的高亮数据为本地文件
- `activeTab`：在你当前操作的标签页上执行高亮与翻译相关功能
- `tabs`：打开历史页、设置页或原网页，并与当前标签页协同工作
- `https://*/*` 主机权限：在你访问的 HTTPS 网页中提供高亮与翻译能力，并访问所需的 HTTPS 接口

这些权限不代表我们会持续、自动或无差别地收集所有网页内容。扩展主要处理的是你主动选中、主动保存、主动翻译或主动同步的数据。

## 6. 数据共享与第三方服务

在以下情况下，数据可能会被共享给第三方服务，但仅限于完成你主动请求的功能：

- 当你使用翻译功能时，相关文本和必要上下文可能被发送给有道翻译或你自己配置的 OpenAI 兼容翻译服务
- 当你使用邮箱验证功能时，邮箱地址和必要邮件内容可能通过事务性邮件服务提供商发送
- 当你访问本扩展配套后端时，数据会经过明彩后端服务处理和存储

除上述情形外，我们不会将你的个人数据出售、出租或无关共享给第三方。

如果法律法规、法院命令或监管要求要求披露数据，我们也可能在法律要求范围内披露必要信息。

## 7. 数据保存期限

不同类型数据的保存期限如下：

- 本地浏览器中的高亮、设置和令牌信息：保存至你主动删除、退出登录、清空数据或卸载扩展为止
- 云同步的高亮数据和账号资料：保存至你主动删除相关内容或注销账号为止
- 邮箱验证令牌：保存至验证完成、过期或被替换为止
- 翻译请求的最小化运行日志：保存至实现安全防护、计费控制、故障排查和运营维护所需的合理期限为止

## 8. 你的选择与权利

你可以通过以下方式控制自己的数据：

- 不注册账号，仅使用本地功能
- 删除本地高亮、导入数据或清空当前页面数据
- 退出登录，清除本地登录状态
- 在设置页中注销账号并删除云端数据
- 联系我们请求访问、更正或删除与你账号相关的数据

如果你是适用法律下的数据主体，并希望行使访问、更正、删除或限制处理等权利，可以通过本政策末尾的联系方式与我们联系。

## 9. 安全措施

我们采取合理的技术和组织措施来保护数据安全，包括但不限于：

- 对账号密码进行哈希存储，而不是明文存储
- 使用访问令牌和刷新令牌维护登录状态
- 通过 HTTPS 与后端和第三方接口通信
- 将翻译请求的部分运行信息以哈希或最小化形式记录，用于防滥用和故障排查

但请理解，任何互联网传输或电子存储方式都无法保证绝对安全。
